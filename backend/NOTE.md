# Anotações Back-end

## KnexJS

- Criação de um migrate

```
    yarn knex migrate:make create_ongs
```

- Run migrate

```
   yarn knex migrate:latest
```

- Desfazer a migrate

```
  yarn knex:rollback
```

## Validações dos dados

- [celebrate](https://github.com/arb/celebrate)

```
  yarn add celebrate
```
